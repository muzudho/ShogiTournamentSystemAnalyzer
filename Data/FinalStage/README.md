# FinalStage

このフォルダーは、本戦専用モードで再利用する補助 CSV を置く場所です。

## 今ある例
- `【グループ対応表】[本戦専用_トップ集団大きめ].csv`
- `【グループ対応表】[本戦専用_トップ集団小さめ].csv`
- `【グループ対応表】[本戦専用_先手8x後手8].csv`
- `【本戦不出場Apex一覧】[先手8x後手8_2名].csv`

## 想定しているデータ
- グループ対応表
- 本戦不出場Apex一覧

例:
- `group_map_[本戦専用_先手8x後手8].csv`
- `group_map_[本戦専用_トップ集団小さめ].csv`
- `additional_apex_[先手8x後手8_2名].csv`

## 位置づけ
- `Inputs`: 実行用入力ファイル
- `Data/Players`: 再利用する選手データ
- `Data/Matches`: 再利用する対局データ
- `Data/FinalStage`: 本戦専用の補助データ
- `Data/RuleSets`: 保存して再利用する大会ルール
