# RuleSets

このフォルダーは、保存して再利用する大会ルール定義を置く場所です。

## 想定している内容
- Neutral / Twill などのルールセット
- ユーザーが保存した大会ルール
- 地域別・年度別のルール定義

例:
- `neutral_default.json`
- `twill_standard.json`
- `2026_kanto_standard.json`

## 位置づけ
- `Data/Inputs`: 実行用入力ファイル
- `Data/Participants`: 再利用する参加者データ
- `Data/RuleSets`: 保存して再利用する大会ルール
