# Examples

このフォルダーは、人が読むための見本を置く場所です。

## 置くもの
- 対局記号表の見本
- 入力形式を理解するための簡単な CSV
- メモ付きの説明用 Markdown

## 置かないもの
- 再利用する選手一覧 CSV
- 再利用する対局表、参考対局表 CSV
- 本戦専用モードのグループ対応表や補助 CSV
- `--input-file` で流す実行用入力ファイル

## 関連する置き場
- `Data/Players`: 再利用する選手データ
- `Data/Matches`: 再利用する対局表、参考対局表
- `Data/FinalStage`: 本戦専用モードの補助データ
- `Inputs`: 実行用入力ファイル
