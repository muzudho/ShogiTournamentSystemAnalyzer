# 【参考対局】[先手8x後手8_本戦不出場Apexあり]

本戦不出場Apexがいるケースで、**参考対局** を追加入力するときのサンプルです。  

## 入力に使うファイル
- `【選手一覧】[先手8x後手8].csv`
- `【グループ対応表】[本戦専用_先手8x後手8].csv`
- `【本戦不出場Apex一覧】[先手8x後手8_2名].csv`
- `【対局表】[先手8x後手8].csv`
- `【参考対局表】[先手8x後手8_本戦不出場Apexあり].csv`

## 参考対局の位置づけ
- 参考対局は **大会記録に含めません**
- 参考対局は **品質評価にも含めません**
- 空席を埋めるための観察用・運営補助用の対局として使います

## 使い方
1. 本戦専用モード、または品質評価モードを開く
2. 公式対局の入力を終える
3. `参考対局CSVまたは Round/First-Second / PlayerSymbols を貼り付けてください。` と表示されたら、
   `【参考対局表】[先手8x後手8_本戦不出場Apexあり].csv` の内容を貼り付ける
4. 入力を省略したいときは、空のまま `END` を入力する

## このサンプルで見たいこと
- 参考対局を入力しても順位計算は変わらないこと
- 参考対局が別CSVとして保存されること
- 運営補助としては記録を残せること
