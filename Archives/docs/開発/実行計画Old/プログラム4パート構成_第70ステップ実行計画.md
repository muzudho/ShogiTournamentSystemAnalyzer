# プログラム4パート構成 第70ステップ実行計画

## 目的

`STSAInput/4` の `Simulation + Standard` で、21局以上かつ `SimulationCount` 指定ありの要求を新ルートで実行する。

## 実装範囲

- `StsaInput4RequestParser` は、標準シミュレーションの20局超を `SimulationCount` 付きなら受け入れる。
- `AnalysisRequestDispatcher` は、`StandardSimulationRequest.SimulationCount` を標準シミュレーション主線へ渡す。
- `StandardSimulationMainline` / `AbstractSimulationMainline` は、指定済みの試行回数を使い、未指定時だけ従来どおりコンソール入力へ戻る。
- 21局の標準シミュレーション smoke 入力を追加する。

## 対象外

- `Simulation + Standard` の20局超で `SimulationCount` 未指定の要求。
- `AnalysisFlowSteps=Simulation,QualityEvaluation` の複数ステップ要求。
