# docs

このフォルダーは、ShogiTournamentSystemAnalyzer の文書置き場です。  
利用者向けの説明書、開発向けの実装計画、実験レポート、発想メモを役割ごとに分けています。

## まずどこを見るか
- 利用方法や仕様を知りたい
  - [Manuals](./Manuals/README.md)
- 実装予定や改修メモを見たい
	- [実装計画](./実装計画/README.md)
- 実験結果や品質評価レポートを見たい
	- [Reports](./Reports/README.md)
- 発想メモやルール案を見たい
  - `むずでょの案`
