## 2026-06-02 追記: プロセス全体への入力副作用は禁止

- `Console.SetIn()` のように、プロセス全体の標準入力を差し替える実装は禁止。
- 要求ファイルを読む場合は、`File.ReadAllLines(path)` や `StsaFileIOHelper.ReadAllLines(...)` のように、ファイル内容を明示的に読み取って処理へ渡す。
- `RequestFileCheckWorkflow` では、要求ファイルチェックのために `Console.In` を差し替えない。
- 後続処理が入力行を必要とする場合は、`RequestInputSession` などのモデルへ読み取り済みデータを持たせるか、明示的な引数として渡す。
- 手動入力と要求ファイル入力は、同じ `Console.ReadLine()` 導線へ無理に合流させず、プロセスグローバル状態に依存しない導線へ分ける。
