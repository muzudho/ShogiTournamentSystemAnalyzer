# プログラム4パート構成 第25ステップ実行計画

## 目的

`ApplicationRun` に残っている、入力セッションを開いて分析を実行し、最後にセッションを完了する処理を分離する。

`ApplicationRun` は「起動準備」と「依頼実行」を並べるだけに寄せ、入力セッション付きの実行単位は `ApplicationRequestRun` にまとめる。

## 対象

- `ShogiTournamentSystemAnalyzer/Application/Execution/ApplicationRun.cs`
- `ShogiTournamentSystemAnalyzer/Application/Execution/ApplicationRequestRun.cs`

## 作業

1. `ApplicationRequestRun` を追加する。
2. 入力元設定、分析実行、入力セッション完了を `ApplicationRequestRun.Run(args)` に移す。
3. `ApplicationRun.Run(args)` は `ApplicationStartup.Start()` と `ApplicationRequestRun.Run(args)` の順にする。
4. 外部挙動は変えない。

## 確認

- `dotnet build .\ShogiTournamentSystemAnalyzer\ShogiTournamentSystemAnalyzer.csproj`
- 空ルールの要求ファイル入力スモーク
- 手動入力保存のスモーク
