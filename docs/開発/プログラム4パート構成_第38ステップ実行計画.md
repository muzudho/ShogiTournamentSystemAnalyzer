# プログラム4パート構成 第38ステップ実行計画

## 目的

複数パートから参照される実行時間予算管理を `Application/Execution/Shared/` へ配置する。

`SimulationTimeBudget` は、起動準備、コンソール入力、シミュレーション、品質評価など複数の実行パートで使われる。単一パートに置くと誤解が増えるため、例外的に `Shared` に置く。

## 対象

- `ShogiTournamentSystemAnalyzer/Application/Execution/SimulationTimeBudget.cs`
- `ShogiTournamentSystemAnalyzer/Application/Execution/Shared/`

## 作業

1. 配置マップを更新する。
2. `SimulationTimeBudget.cs` を `Application/Execution/Shared/` へ移動する。
3. namespace は今回は維持し、外部挙動を変えない。
4. `#メモ：` や `#ここは：` で始まるユーザー用メモは触らない。

## 確認

- `dotnet build .\ShogiTournamentSystemAnalyzer\ShogiTournamentSystemAnalyzer.csproj`
- 空ルールの要求ファイル入力スモーク
- 手動入力保存のスモーク
