# プログラム4パート構成 第105ステップ実行計画

## 目的

TournamentFramework シミュレーションの出力呼び出しで、最終順位付け結果に関する引数を `TournamentFrameworkFinalRankingResult` へまとめる。

## 背景

第104ステップで、標準形式プレイヤー/対局表、aggregate 計算結果、aggregate 最終順位表の組み立てを `FinalRankingDomain.BuildTournamentFrameworkFinalRankingResult(...)` へ移した。

ただし `FinalRankingDomain.WriteTournamentFrameworkSimulationOutputs(...)` は、呼び出し元から representative 順位行、aggregate 計算結果、aggregate 順位行を個別に受け取っていた。これは `TournamentFrameworkFinalRankingResult` と同じ情報を分解して渡しており、呼び出し元に最終順位付け結果の内部構造を余分に意識させていた。

## 実施内容

- `WriteTournamentFrameworkSimulationOutputs(...)` の引数を整理し、最終順位付け結果のまとまりを `TournamentFrameworkFinalRankingResult` で受け取る。
- aggregate CSV / Markdown と representative CSV / Markdown の生成では、受け取った結果オブジェクトのプロパティを参照する。
- `SimulationTournamentFrameworkMode` の出力呼び出しを、個別プロパティ渡しから結果オブジェクト渡しへ変更する。
