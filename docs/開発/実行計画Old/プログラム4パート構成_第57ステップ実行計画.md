# プログラム4パート構成 第57ステップ実行計画

## 目的

`InputSourceConfiguration` の責務を、入力導線の選択に絞る。

要求ファイルを実際にチェックして、成功時に入力セッションへ変換する処理は `RequestFileInputSessionStarter` へ分離する。

## 対象

- `Application/BeforeRequestFileCheck/InputSourceConfiguration.cs`
- `Application/BeforeRequestFileCheck/RequestFileInputSessionStarter.cs`

## 作業

1. `RequestFileInputSessionStarter` を追加する。
2. 要求ファイルチェック、入力適用、入力セッション作成を `RequestFileInputSessionStarter.TryStart()` へ移す。
3. `InputSourceConfiguration` は、引数読み取り結果に応じて要求ファイル入力か手動入力かを選ぶ役に絞る。
4. 挙動は変えず、責務分離だけに限定する。

- 空ルールの要求ファイル入力スモーク
- 手動入力保存のスモーク
