# プログラム4パート構成 第101ステップ実行計画

## 目的

標準/本戦シミュレーション mainline から最終順位表の CSV / Markdown 書き出し責務を外し、出力処理を `FinalRankingDomain` 経由へ寄せる。

## 今回の範囲

- `FinalRankingDomain` に標準シミュレーション用と本戦シミュレーション用の出力入口を追加する。
- `StandardSimulationMainline` / `FinalStageSimulationMainline` から出力パス状態、writer 生成、CSV / Markdown 書き出し helper を削除する。
- `SimulationDomain` が mainline 結果を受け取った後、`FinalRankingDomain` へ最終順位表出力を依頼する。
- 旧対話シナリオの `StandardSimulationScenario` / `FinalStageSimulationScenario` も、mainline 実行後に同じ出力入口を呼ぶ。

## 今回は変えないこと

- シミュレーション計算ロジック。
- 最終順位行の内容。
- CSV / Markdown writer の出力フォーマット。
- TournamentFramework / Empty シミュレーションの出力整理。
