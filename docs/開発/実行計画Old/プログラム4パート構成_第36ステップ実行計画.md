# プログラム4パート構成 第36ステップ実行計画

## 目的

入力導線の初期分岐を、ライフサイクル上の `BeforeRequestFileCheck` フォルダーへ配置する。

`InputSourceConfiguration` は、コマンドライン引数から要求ファイル指定を読み、要求ファイルチェックへ進むか、手動入力へ進むかを決める。これは要求ファイルチェック前の分岐にあたるため、`Application/Execution/BeforeRequestFileCheck/` に置く。

## 対象

- `ShogiTournamentSystemAnalyzer/Application/Input/InputSourceConfiguration.cs`
- `ShogiTournamentSystemAnalyzer/Application/Execution/BeforeRequestFileCheck/`

## 作業

1. 配置マップを更新する。
2. `InputSourceConfiguration.cs` を `Application/Execution/BeforeRequestFileCheck/` へ移動する。
3. namespace は今回は維持し、外部挙動を変えない。
4. `#メモ：` や `#ここは：` で始まるユーザー用メモは触らない。
