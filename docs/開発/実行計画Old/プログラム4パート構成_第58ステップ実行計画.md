# プログラム4パート構成 第58ステップ実行計画

## 目的

namespace とクラス名の重複を減らし、alias using が不要な名前へ整える。

`ManualInput.ManualInput` と `RequestFileCheck.RequestFileCheck` は、フォルダー名とクラス名が同じで読みづらいため、責務が分かるクラス名へ変更する。

## 対象

- `Application/ManualInput/ManualInput.cs` -> `Application/ManualInput/ManualInputSessionStarter.cs`
- `Application/RequestFileCheck/RequestFileCheck.cs` -> `Application/RequestFileCheck/RequestFileChecker.cs`
- `Application/BeforeRequestFileCheck/InputSourceConfiguration.cs`
- `Application/BeforeRequestFileCheck/RequestFileInputSessionStarter.cs`

## 作業

1. `ManualInput` class を `ManualInputSessionStarter` へリネームする。
2. `RequestFileCheck` class を `RequestFileChecker` へリネームする。
3. alias using を通常の using とクラス参照へ置き換える。
4. 挙動は変えず、命名整理だけに限定する。
