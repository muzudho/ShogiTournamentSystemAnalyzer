# プログラム4パート構成 第60ステップ実行計画

## 目的

`BeforeManualInput` が未着手のまま残っているため、手動入力へ入る直前の案内とフォールバック分岐を配置する。

`InputSourceConfiguration` は要求ファイル入力か手動入力かを選ぶ役に絞り、手動入力へ切り替えるときの表示と開始準備を `BeforeManualInput` 側へ寄せる。

## 対象

- `Application/BeforeManualInput/ManualInputSessionPreparation.cs`
- `Application/BeforeRequestFileCheck/InputSourceConfiguration.cs`
- `Application/ManualInput/ManualInputSessionStarter.cs`
- `docs/開発/プログラム4パート構成_配置マップ.md`

## 作業

1. `BeforeManualInput` に `ManualInputSessionPreparation` を追加する。
2. 引数エラー、要求ファイル指定なし、要求ファイルチェック失敗後の手動入力開始を `ManualInputSessionPreparation` に集める。
3. `ManualInputSessionStarter` は手動入力セッションの開始本体に絞る。
4. 挙動は変えず、表示と分岐の置き場だけを整理する。
