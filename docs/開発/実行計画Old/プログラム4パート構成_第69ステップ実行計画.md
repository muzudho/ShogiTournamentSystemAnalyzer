# プログラム4パート構成 第69ステップ実行計画

## 目的

`STSAInput/4` の `Simulation + FinalStage` で、21局以上かつ `SimulationCount` 指定ありの要求を新ルートで実行する。

## 実装範囲

- `StsaInput4RequestParser` は、本戦シミュレーションの20局超を `SimulationCount` 付きなら受け入れる。
- `AnalysisRequestDispatcher` は、`FinalStageSimulationRequest.SimulationCount` を本戦シミュレーション主線へ渡す。
- `FinalStageSimulationMainline` / `FinalStageSimulationExecutor` は、指定済みの試行回数を使い、未指定時だけ従来どおりコンソール入力へ戻る。
- 21局の本戦シミュレーション smoke 入力を追加する。

## 対象外

- `Simulation + Standard` の20局超直通化。
- `AnalysisFlowSteps=Simulation,QualityEvaluation` の複数ステップ要求。
