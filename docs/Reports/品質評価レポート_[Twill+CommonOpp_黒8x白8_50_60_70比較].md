# 品質評価レポート: Twill+CommonOpp の途中比較（黒8x白8 / 50%・60%・70%）

## 概要
このレポートは、新しく追加した **Twill+CommonOpp** を、既存の **Neutral** / **Twill** と比較した途中まとめです。

対象ケースは `黒8x白8` で、先手勝率 `50% / 60% / 70%` の比較を行いました。

ただし、`Twill+CommonOpp` は計算時間の都合でスイープを最後まで完走できず、今回は次の 3 点までを比較対象にしています。

- 50%
- 60%
- 70%

- 比較対象ルール:

## 見えてきたこと

### 1. 50% ではかなり有望
`Twill+CommonOpp` は 50% でかなり良い値を出しています。

- Spearman は `1.000000`
- 平均順位ずれは 3 ルール中で最良
- 上位8残留も `Neutral` とほぼ同等

この時点では、**Twill の良さを残しつつ、全体順位の安定感を少し整えた** ように見えます。

### 2. 60% ではトレードオフが見え始める
60% では、`Twill+CommonOpp` の性格が少し見えてきます。

良い点:
- 平均順位ずれは `Twill` よりわずかに良い
- 上位8残留は 3 ルール中で最良

気になる点:
- Spearman は `Neutral` / `Twill` より下がる
- Elo1位総合1位確率は大きく落ちる

つまり、**全体のならし方は良いが、最強者を鋭く押し上げる力は弱まりやすい** 可能性があります。

### 3. 70% では崩れ始めるが、見方は単純ではない
70% では `Twill+CommonOpp` もかなり苦しくなります。

- Spearman は `Twill` より少し悪い
- 平均順位ずれも `Twill` より少し悪い
- ただし上位8残留は `Twill` とほぼ同等
- Elo1位総合1位確率は `Twill` よりかなり高い

ここは単純な優劣というより、

- `Twill`: 全体順位の並びや平均順位は比較的きれい
- `Twill+CommonOpp`: 最強者の押し上げは少し戻る

という **評価軸の違い** が出ているように見えます。

## 途中結論
現時点の途中結論は次の通りです。

- `50%` では `Twill+CommonOpp` はかなり有望
- `60%` では平均順位ずれと上位8残留は良いが、最強者の押し上げが弱い
- `70%` では `Twill` に対する明確な上位互換とは言いにくい

したがって、今の `Twill+CommonOpp` は

**「Twill をそのまま強化した完成形」ではなく、低い先手勝率帯で光る別バランス案**

として見るのが自然です。

## 今後の詰めどころ
次に見るなら、次のどちらかが有効です。

1. `80% / 90% / 100%` もさらに軽量条件で取り切る
2. `Twill+CommonOpp` の信頼度係数や閾値を調整して第2版を試す

特に今回の結果を見る限り、

- 50% の良さを残したい
- 60% 以降の Spearman 低下と Elo1位総合1位の落ち込みを和らげたい

という方向で調整するのがよさそうです。

## まとめ
今回の途中比較からは、`Twill+CommonOpp` に次の特徴が見えます。

- 低い先手勝率帯ではかなり良い
- 中間帯では平均順位の安定感と最強者押し上げの両立がまだ難しい
- `Twill` の単純上位版というより、別のクセを持つ派生案である

よって現時点では、

- **有望な改良候補ではある**
- **ただし、まだ係数調整や再実験の余地が大きい**

という整理が妥当です。
