# モード別ガイド

## モード一覧
起動後、まず目的を選び、その後に対象ルールを選びます。

補足として、主要な入力は誤りが 10 回続くと中断します。入力ファイルを使う場合も同じです。

- 目的
  - `1`: 対局シミュレーション
  - `2`: 品質評価
- 対象ルール
  - `1`: 通常ルール
  - `2`: 本戦ルール

## 対局シミュレーション / 通常ルール
対局シミュレーション / 通常ルールでは、総当たり戦をフラットに分析します。

### 入力順
1. 順位ルールセット（`Neutral / Twill`）
2. 先手勝率 (%)
3. 選手 / Player 一覧CSV
4. 対局CSV または Round/First-Second/対局記号表
5. 必要に応じてシミュレーション試行回数
6. 結果CSVの出力先パスまたはフォルダーパス

### 向いている用途
- 総当たり戦の公平性確認
- 対局表による有利不利の比較
- `Neutral` と `Twill` の基本比較

## 対局シミュレーション / 本戦ルール
対局シミュレーション / 本戦ルールでは、Apex / Innov を使う定先戦向けの分析を行います。

### 入力順
1. 同Elo対局時の先手勝率 (%)
2. 選手 / Player 一覧CSV
3. Apex / Innov の分け方（On / Off）
4. `Off` のときだけ順位ルールセット（`Neutral / Twill`）
5. `On` のときだけグループ対応CSV
6. `On` のときだけ本戦不出場Apex一覧CSV（省略可）
7. `On` のときだけ本戦不出場Apexの扱い（Off / On）
8. `On` のときだけ境界救済戦の有無（Off / On）
9. 対局CSV または Round/Black-White/対局記号表
10. 参考対局CSV または Round/Black-White/対局記号表（省略可）
11. 必要に応じてシミュレーション試行回数
12. 結果CSVの出力先パスまたはフォルダーパス

### 前提
現状の MVP 実装では、次を前提にしています。

- 本戦選手は 16 名
- Apex / Innov 分け方が `On` のときは `Apex` は 8 名以下
- Apex / Innov 分け方が `On` のときは対局は `Innov` が黒番、`Apex` が白番
- Apex / Innov 分け方が `On` のときは総合順位は、まず本戦出場 `Apex` 内で順位づけし、その後ろに本戦出場 `Innov` を並べる
- Apex / Innov 分け方が `Off` のときは、`Neutral` または `Twill` を選んで全選手をフラットに順位づけする
- 本戦不出場Apexと境界救済戦は、Apex / Innov 分け方が `On` のときだけ使う

## 品質評価 / 通常ルール
品質評価 / 通常ルールでは、総当たり戦向けルールの実力反映性を評価します。

### 入力順
1. 順位ルールセット（`Neutral / Twill`）
2. 同Elo対局時の先手勝率 (%)
3. 選手 / Player 一覧CSV
4. 対局CSV または Round/Black-White/対局記号表
5. 単発評価または n% スイープ実験の選択
6. 必要に応じてシミュレーション試行回数
7. レポート出力先パスまたはフォルダーパス

## 品質評価 / 本戦ルール
品質評価 / 本戦ルールでは、本戦ルールの実力反映性を評価します。

### 入力順
1. 同Elo対局時の先手勝率 (%)
2. 選手 / Player 一覧CSV
3. Apex / Innov の分け方（On / Off）
4. `Off` のときだけ順位ルールセット（`Neutral / Twill`）
5. `On` のときだけグループ対応CSV
6. `On` のときだけ本戦不出場Apex一覧CSV（省略可）
7. `On` のときだけ本戦不出場Apexの扱い（Off / On）
8. `On` のときだけ境界救済戦の有無（Off / On）
9. `On` のときだけ可変定員8ルールの有無（Off / On）
10. 対局CSV または Round/Black-White/対局記号表
11. 参考対局CSV または Round/Black-White/対局記号表（省略可）
12. 単発評価または n% スイープ実験の選択
13. 必要に応じてシミュレーション試行回数
14. レポート出力先パスまたはフォルダーパス

### 補足
- 品質評価 / 通常ルールでは `Neutral / Twill` を切り替えられます
- 品質評価 / 本戦ルールでは Apex / Innov の分け方、境界救済戦、可変定員8ルールなどを評価対象にできます
- 品質評価の詳細は [品質評価](./品質評価.md) を参照してください
