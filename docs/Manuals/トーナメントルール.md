# トーナメントルール

## 前提ルール
- 各対局は黒番・白番を持つ
- 同 Elo 対局時の黒番勝率を 0〜100% で指定する
- 勝率は Elo 差と黒番補正から計算する
- 同点順位は該当順位帯に等分配する

## 順位ルールセット
このアプリでは、順位づけのルールセットとして `Neutral` と `Twill` を扱えます。

## Neutral
- 既定の順位ルールです
- 各シナリオでの勝ち数をもとに順位を決めます
- 同点順位は該当順位帯に等分配します

### 向いている用途
- 基準ルールとしての比較
- 単純な勝ち数順位の確認
- 品質評価でのベースライン

## Twill
- **ツイル式トーナメント** の順位ルールです
- 各対局結果を `▲ / ▽` の比較グラフとして読み、最後に **重箱表** 相当の指標へたたんで順位を決めます

### 優先順
概ね次の優先順で比較します。

1. 合計右ノード数
2. 合計勝ち星
3. 合計最長経路長
4. `▽` 側で負けた相手の強さ
5. 直接対戦

### 向いている用途
- 単純な勝ち数だけでは見えにくい対戦構造の反映
- 順位づけ規則の比較実験
- `Neutral` との差分観察

## どのモードで選べるか
- 通常モードでは常に `Neutral / Twill` を選べます
- 本戦専用モードと品質評価モードでは、`Apex / Innov` の分け方を `Off` にしたときだけ `Neutral / Twill` を選べます
