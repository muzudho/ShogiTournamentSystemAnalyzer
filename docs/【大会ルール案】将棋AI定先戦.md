# 【大会ルール案】将棋AI定先戦

予選と、本戦に分かれているぜ（＾～＾）  
とりあえず、本戦の説明からするぜ（＾～＾）  


## 背景

その前に背景から（＾～＾）  
将棋AI（コンピューター将棋ソフト）は、Elo レーティングが 4300 を超えてきた頃から、  
先手番の勝率が 60% を超えるようになってきたぜ（＾～＾）  

2026年現在、トップクラスの将棋AIは先手番の勝率が 70% を超えるようになってきたぜ（＾～＾）  
これでは、先手番が有利すぎて、対局の公平性が損なわれているぜ（＾～＾）  

そこで、先手番勝率はこれからどんどん上がっていくことが予想されるので、それを前提とした大会ルールを考えてみたぜ（＾～＾）  


## 本戦

二次予選で、先手番を持って負けなかったソフトを、［Apex グループ］と呼ぶとするぜ（＾～＾）
それ以外のソフトを［Innov グループ］と呼ぶとするぜ（＾～＾）  

本戦に進めるのは１６ソフトだぜ（＾～＾）  
［Apex グループ］の上位から８ソフトが本戦に進み、▽後手番を持つぜ（＾～＾）  
［Innov グループ］の上位から８ソフトが本戦に進み、▲先手番を持つぜ（＾～＾）  

特別ルールによっては、［フロムスクラッチ１位］のソフトや、［ノートＰＣ１位］のソフトが［Innov グループ］として  
本戦に進む枠が設けられることも考えられるぜ（＾～＾）  


総合順位は、［Apex グループ］の順位の後に［Innov グループ］の順位が続くぜ（＾～＾）
例えば、［Apex グループ］の１位が全体の１位、［Apex グループ］の２位が全体の２位、…、［Apex グループ］の８位が全体の８位、
そして、［Innov グループ］の１位が全体の９位、［Innov グループ］の２位が全体の１０位、…、［Innov グループ］の８位が全体の１６位になるぜ（＾～＾）

［Apex グループ］が８ソフト未満の場合は、残りの枠を［Innov グループ］の上位から埋めるぜ（＾～＾）  
［Apex グループ］が８ソフトを超える場合は、［Apex グループ］の９ソフト以降は本戦に進めないが、  
総合順位は９位から順番にもらえるぜ（＾～＾）  


## 予選

先手と後手がなるべく同じ数になるような対戦表を動的に組まれるぜ（＾～＾）
先手番を持って負けなかったソフトが［Apex グループ］に入るんで、先手番ではぜったい負けたくないぜ（＾～＾）  
最近は、同じ強さのソフト同士なら後手番で勝ちにくくなっているぜ（＾～＾）  


## シミュレーションしたい

ここで、このルールで大会をやったら、どんな順位分布になるのかシミュレーションしてみたいぜ（＾～＾）  
例えば、Elo レーティングの上位１０ソフトが団子状態で、下の集団と差をつけている場合や、  
上位ソフト２つが抜けていて、３位以下が団子状態になっている場合など、いろんなパターンでシミュレーションしてみたいぜ（＾～＾）
