## １ラウンドごとにグラフを作っていく

ホワイトボードに結果を、指向性グラフとして記録する。  
▲竜　と　▽竜　は別のノードとして扱う。  

以下、単純にレーティングが高い方が勝つと仮定した場合の例である。  

結果表（左が負け、右が勝ち）：  

```
▽ひよこ　　▲らいおん
▽いのしし　▲きりん
▽うさぎ　　▲ぞう
▽ねこ　　　▲いぬ
```

結果グラフ（右の方に勝者が来る指向性グラフ）：  

```
▽ひよこ　─→▲らいおん
▽いのしし─→▲きりん
▽うさぎ　─→▲ぞう
▽ねこ　　─→▲いぬ
```
