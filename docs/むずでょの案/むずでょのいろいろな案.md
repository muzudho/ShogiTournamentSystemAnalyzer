# むずでょのいろいろな案


## ［アプリケーション・フロー］

* ［◆大会ルールデータ］、［◆プレイヤー一覧データ］、［◆順位付けの設定データ］を作成、またはファイルから読込
* ［シミュレーション］フロー
* ［順位付け］フロー
* ［大会品質評価］フロー


### ［シミュレーション］フロー

* ［◆大会ルールデータ］、［◆プレイヤー一覧データ］を入力
* ［シミュレーション］開始
    * 都度、［中間順位付け］開始
    * ［中間順位］出力
* ［◆大会結果データ］出力


### ［順位付け］フロー

シミュレーション中にやれば［中間順位付け］  
シミュレーション後にやれば［最終順位付け］  

* ［◆順位付けの設定データ］、［◆大会結果データ］入力
* ［順位付け］開始
* ［◆最終順位データ］出力


### ［大会品質評価］フロー

* ［◆最終順位データ］（効果的なら［◆プレイヤー一覧データ］、［◆大会ルールデータ］も）入力
* ［大会品質評価］開始
* ［◆大会品質レポート］出力
