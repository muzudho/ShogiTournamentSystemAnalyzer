# ツイル式トーナメントの普及のために（スタッフ用文章）


## お客さんが欲しがっているもの

* 専門的な言い方をすると、コミの無い囲碁でも、強い順に順序が付くトーナメント方式。
* 先手勝率７０％以上の対戦ゲームで、先手を取った数と後手を取った数がいくつであっても、結果の順位はそう変わらないゲームであってほしい。
* 先手勝率１００％なら仕方ない。それは除く。
* 大会前の Elo を知らなくても、大会中だけでマッチングが適切に決まるのは嬉しい。
* ルールを文章で読むのは難しいので、例を見て理解できると嬉しい。
* 統計データは、（無いより有る方が良いが）別紙としてまとめてほしい。文章の中に入れると、理解の妨げになる。
