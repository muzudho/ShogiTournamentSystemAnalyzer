## 命名メモ

- 大会名: **ツイル式トーナメント**
- 英語名: **Twill Tournament**
- 順位を読む集計表: **重箱表**

`Twill` は織物の斜めに流れる織り模様で、  
`▲/▽` ノードが交差しながらつながる図のイメージに近い。
