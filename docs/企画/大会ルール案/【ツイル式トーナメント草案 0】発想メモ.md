# 【ツイル式トーナメント草案 0】発想メモ

※ 旧題: `【大会ルール案】格付けグラフ戦`  
ツイル式トーナメントへ整理する前の、最初期の発想メモ。

（１）先手を持って負けたら、（自分に勝った）相手の名前を［名札］にして持っておく。  

（２）次のラウンド開始前に、ホワイトボードに指向性グラフにする。  
名札を持っている人から、名札の人へ矢印を引く。  

```
（例）  
自分→いぬ  
ぞう→ねこ  
```

（３）何ラウンドか進むと、グラフができる。

```
（例）
自分→いぬ─┐
　　　　　　├→ねこ
　　　ぞう─┘

うさぎ→桂
```

（４）つながっていないグラフがあれば、優先して対局を付ける。  
上図の場合、うさぎと桂はつながっていないので、うさぎと桂を優先して対局させる。

（５）なるべく、１つのグラフができるまで続ける。

（６）グラフの中で、指されている人の方が、順位が高いとみなす。

（７）同じ順位の人の中では、勝ち星が多い人の方が、順位が高いとみなす。

（８）それでも同じ順位の人がいる場合は、同順とする。

```
（例）
　　自分→いぬ─┐
　　　　　　　　├→ねこ
　　　　　ぞう─┘　↑
　　　　　　　　　　│
　　　　　　　　　　↓
　　　　　　うさぎ→桂
```

（９）もしかすると、双方向の矢印ができるかもしれない。

（１０）もしかすると、循環するかもしれない。
