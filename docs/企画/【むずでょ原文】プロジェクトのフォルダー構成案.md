# 【むずでょ原文】プロジェクトのフォルダー構成案

## 互換性用フォルダー

AIエージェントは互換性のロジックを残したがるので、それ用のフォルダーを置いておいてやりたい（＾～＾）　（AIエージェントの飼い方）  

```plaintext
📁 {プロジェクトフォルダー}
+-- 📁 Compatibility
	+-- 📁 {Legacy*}
	+-- 📁 {Internal*}
	*-- 📄 *Facade.cs
```
* `Legacy`：古いコードを残すためのフォルダー。将来的に削除する予定のコードをここに移動する。
* `Internal`：内部で使用するコードを格納するフォルダー。外部に公開しないコードをここに配置する。
* `Facade.cs`：プロジェクトの主要なインターフェースを提供するファサードクラス。外部からのアクセスをここで管理する。
